A simple Yackstar stream plugin.

== Description ==

This is a simple plugin for wordpress that allows you to display your Yackstar Mini Stream.

== Installation ==

The easiest way to use Yackstar Stream is to add it via the widgets.
Just go to the widgets page and add the Yackstar Stream widget to one of your widget areas.
The alternative is to use the function by including <?php yackstar_stream(); ?> in your template somewhere.
